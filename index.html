Introduction to Modern Web Development

A Primer

This set of ... articles(?) isn't intended to be a deep dive in to any specific technologies, languages, tools, etc. Nor am I going to often add my opinion or a value statement, instead I'll try to focus on popularity first, and my own familiarity when I want to fill out more relevant content on subjects that may not have a popular concensus.

Modern web development is comprised of three main building blocks:

  • HTML (HyperText Markup Language)
  • Javascript (JS)
  • CSS (Cascading style sheets)

HTML HTML: HyperText Markup Language on MDN

HTML is the language that defines the content, and can even provide some semantic meaning to that content through the use of containers call tags or elements.

Here is an example of of some HTML code:



Hello World!

Hello World!

The above code tells the browser to display the text "Hello World!" as a paragraph. You can program things like <article> for writing, <canvas> for drawing, and <video> for, well, video!

JavaScript JavaScript on MDN

JavaScript is a language that complements HTML and CSS by bringing behaviour, logic and dynamic interaction capabilities to what would otherwise be a very static experience. HTML or CSS don't typically handle scenaios like if a user clicks on a button declared in HTML, pop up an alert that says "Hello World!". This is where Javascript can add a lot to a web experience.

In this demo you can see HTML declaring a <script> tag and a <button class="helloWorld"> tag.

Inside the <script> there is some JavaScript, that basically does the following:

  1. window.addEventListener("DOMContentLoaded", function () { ... }); says run the code inside the function curly braces ({}) after the document has loaded (DOMContentLoaded)
  2. button = document.querySelector("aside button.helloWorld"); says find in the HTML a <button class="helloWorld"> tag inside an <aside"> tag, and store it in a container called "button".
  3. button.onclick = function() {alert("Hello World!");}; says when you click the button show an alert dialog that says "Hello World!" :)

CSS CSS: Cascading Style Sheets on MDN

CSS allow a developer to make the content of the HTML look good. Using a "selector" (a way of targeting HTML elements/tags) and then providing a set of rules describing how the content should look.

Here is an example:




This is FUNKY!

This is FUNKY!

So what's happening here? Well:

  1. Two HTML tags are created to hold our content, a <style> tag to hold our CSS rule, and a <p> (paragraph) tag to hold some text.
  2. In the <style> tag we have a CSS selector p.funky targeting any <p class="funky"> tag with a class called "funky", and tells the browser to add the following CSS properties (contained in the curly braces):
    • border: 5px dashed magenta; gives us a fat dashy border on the paragraph that's bright pink!
    • background-color: yellow; adds a snappy yellow background colour
    • font-size: 30px; set the size of the text to 30 pixels, but you can use many size descriptions ('pt' for points, 'cm' for centimeters, '%' for a percent of the current size)
    • transform: skewX(-20deg); skews the whole paragraph so that it leans to the right

Conclusion

This primer is meant to introduce or reacquaint you with the three main building blocks of the web. Browser use these three languages for define, display and interact with almost all of the content on the web today.
